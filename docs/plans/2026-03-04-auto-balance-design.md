# Auto-Balance Typography Feature Design

## Overview

An auto-balance feature that optimizes line-height, letter-spacing, word-spacing, and font-weight for each typographic element based on real font metrics. Inspired by the TypeBalance Figma plugin but implemented as a formula-based engine with runtime font parsing.

## Architecture

### Font Metrics Engine (`src/lib/font-metrics.ts`)

```ts
interface FontMetrics {
  unitsPerEm: number;
  xHeight: number;     // normalized 0-1 (xHeight / unitsPerEm)
  capHeight: number;    // normalized 0-1
  ascender: number;     // normalized 0-1
  descender: number;    // normalized 0-1 (negative)
}
```

Resolution order:
1. In-memory `Map<string, FontMetrics>` session cache
2. Bundled lookup table (`src/data/font-metrics-table.ts`) — ~100 fonts covering all preset stack fonts + top popular Google Fonts
3. Runtime parse: fetch `.woff2` from Google Fonts CDN, parse with `opentype.js`, extract metrics, cache
4. Fallback defaults by font category (sans-serif, serif, display, mono, handwriting)

### Bundled Metrics Table (`src/data/font-metrics-table.ts`)

Pre-computed metrics for ~100 fonts:
- All ~65 unique fonts used across the 38 preset stacks (Abril Fatface, Inter, Playfair Display, Lato, etc.)
- Top ~35 additional popular Google Fonts

Generated by `scripts/generate-font-metrics.ts` — a one-time build script using opentype.js.

### Auto-Balance Formulas (`src/lib/auto-balance.ts`)

Function signature:
```ts
function computeAutoBalance(
  metrics: FontMetrics,
  fontSize: number,
  baseFontSize: number,
  scaleRatio: number,
  isDarkMode: boolean,
  isUppercase: boolean,
): { lineHeight: number; letterSpacing: number; wordSpacing: number; suggestedWeight: number }
```

**Line-height:** Inversely proportional to font size, adjusted by x-height ratio.
```
baseLineHeight = 1 + (0.6 / (fontSize / baseFontSize))
xHeightAdjust = metrics.xHeight * 0.15
lineHeight = baseLineHeight + xHeightAdjust
Clamp: headings 1.0–1.3, body 1.4–1.8
```

**Letter-spacing:** Negative for large text, positive for small, scaled by cap-height.
```
sizeRatio = fontSize / baseFontSize
letterSpacing = -0.02 * Math.log2(sizeRatio) * (metrics.capHeight / 0.7)
Extra tightening for wide scale ratios (>1.333)
Clamp: -0.05em to 0.05em
```

**Word-spacing:** Subtle tightening for large headings.
```
wordSpacing = -0.01 * Math.max(0, Math.log2(sizeRatio) - 1)
Clamp: -0.03em to 0em
```

**Font-weight optical adjustment:** Dark backgrounds make text appear bolder.
```
weightDelta = isDarkMode ? -100 : 0
suggestedWeight = clamp(baseWeight + weightDelta, 100, 900)
```

**Uppercase compensation:** Extra tracking for all-caps text.
```
if (uppercase) letterSpacing += 0.04 * metrics.capHeight
```

### Store Integration (`src/store/typography-store.ts`)

- New state: `autoBalance: boolean` (default `false`)
- New action: `setAutoBalance(enabled: boolean)`
- Toggle ON: compute balanced values for all 11 elements, apply as element overrides
- Toggle OFF: clear auto-balance overrides, restore group defaults
- Manual overrides take priority — user edits are preserved

### Auto-Balance Hook (`src/hooks/use-auto-balance.ts`)

Reactive hook that watches:
- `autoBalance` toggle state
- Heading and body font families
- Scale ratio and base font size
- Background color (for dark mode detection)

When any dependency changes and auto-balance is ON, recomputes and applies values via `setElementOverride()`.

### Header Button (`src/components/layout/header.tsx`)

- Toggle button between swap/random group and undo/redo group
- Icon: `Wand2` from lucide-react
- Ghost when off, secondary/highlighted when on
- Tooltip: "Auto Balance"

## UX Flow

1. User clicks "Auto Balance" toggle ON
2. Font metrics fetched/loaded (instant for bundled fonts, brief load for others)
3. `computeAutoBalance()` runs for all 11 elements (display-1 through small)
4. Results applied as element overrides
5. Preview updates reactively
6. User can still manually tweak elements — manual overrides preserved
7. Changing fonts or scale ratio re-triggers auto-balance
8. Toggle OFF clears overrides, restores defaults

## Properties Adjusted

| Property | Adjusted | Notes |
|----------|----------|-------|
| Line-height | Yes | Inversely proportional to size, x-height adjusted |
| Letter-spacing | Yes | Negative for headings, positive for body, cap-height adjusted |
| Word-spacing | Yes | Subtle tightening for large text |
| Font-weight | Yes (suggestion) | Optical compensation for dark/light backgrounds |
| Uppercase tracking | Yes | Extra letter-spacing for all-caps |
| Font family | No | User-controlled |
| Font size | No | Determined by scale ratio |
| Colors | No | User-controlled |
